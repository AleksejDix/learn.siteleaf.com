Using the v2 Gem

We recommend using GitHub sync with v2, but you can still use the Gem if that’s your preferred workflow.

The v1 gem is not compatible with v2 sites (and vice versa), so we recommend using a Gemfile to allow both versions to be installed.

Important note: when using push/pull in v2, both theme and content files are included. Always make sure to do a pull first to avoid overwriting your Siteleaf content on push. Your local directory must have a _config.yml in order to push.
