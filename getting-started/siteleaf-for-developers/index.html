Siteleaf for Developers

Siteleaf aims to be a content management system that bridges the gap between easy to develop for and easy to use for clients. Static site generators are great for developers: they’re wicked fast, work easily with version control, and are much easier to set up compared to other content management systems. But when it comes to people that are less tech-saavy, static site generators are difficult to use. They often require knowledge of the command line, and text editors aren’t always the easiest programs to write content in.

By offering an easy to use interface for clients, Siteleaf gives you the ability to use a static site generator, despite your client’s tech. We are fully integrated with Jekyll, a popular static site generator that GitHub uses to power over half a million websites. You can enjoy the benefits of a large open source community — there are thousands of open source Jekyll themes, or you can build your own.

Templating

Jekyll uses Liquid, a template language created by Shopify. Liquid makes it incredibly easy to create themes.

GitHub Sync

Siteleaf also offers GitHub Sync. With this enabled, any changes that are made to the site’s content on Siteleaf are committed and pushed to your site’s GitHub repository. Conversely, if you commit changes to the site on GitHub, those changes will be reflected on Siteleaf.

Hosting

Hosting is incredibly easy to set up with Siteleaf. You can choose between pushing straight to GitHub Pages, Amazon S3, Rackspace Cloud Files, or to your own server with FTP/SFTP.

Plugins

Jekyll and GitHub Pages support a variety of external plugins. Siteleaf supports the same subset of plugins that GitHub supports.

API

We also have a robust API that allows you to integrate Siteleaf with your own site or app. With the API, you can do everything the interface can do, from publishing sites to editing content.
